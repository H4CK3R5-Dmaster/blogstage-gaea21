La Formation REACT JS

  • Seif FEKAIER

La formation REACT JS se faisait depuis des vidéos youtube créé par le développeur Grafikart. Le REACt est une bibliothèque du langage Javascript libre qui a été développée par le célébre Facebook depuis 2013. Son but est principalement de faciliter la création d'une application web monopage en créant des composants dépendant d'un état et générant une page HTML à chaque changement d'état.

Pour moi ce fut la première fois que je développe en REACT JS et ce fut pas compliqué de comprendre comment fonctionne ce langage. chaque jours j'étais contrôlé par le coordinateur afin qu'il puisse voir ma progression durant la formation REACT et il constata que j'ai pas de mal à bien comprendre et que j'arrive à bien géré le react

Après 3 semaines consécutive d'appel et de formation REACT je pu enfin terminé la formation REACT JS et pu montré ma nouvelle compétence au coordinateur afin de poursuivre sur la formation Symfony qui durera que 1 ou 2 semaines.
